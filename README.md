# festapp.net

Static marketing site for [Festapp](https://festapp.net) — mobile app for festivals, conferences, and events.

Built with Next.js, Tailwind CSS, and deployed to Cloudflare Pages.

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npx wrangler pages deploy out --project-name festapp-net
```

## Tech Stack

- **Next.js 16** — static export
- **Tailwind CSS v4**
- **TypeScript**
- **Cloudflare Pages** — hosting
